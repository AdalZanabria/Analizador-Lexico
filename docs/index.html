Analizador Léxico

Lenguaje
  • El alfabeto permitido son las cadenas que incluyen las letras: "a", "b", "c", "d" y "e". Éstas solo pueden ser minúsculas.
  • Escriba una cadena para validarla.
  • Use la instrucción U(cadena1,cadena2) para concatenar dos cadenas.
  • Use la instrucción N(cadena,numero) para mostrar el caracter en el lugar numero de la cadena. El conteo inicia con el número 0.
  • Use la instrucción L(cadena) para mostrar la longitud de la cadena.
  • Use la instrucción P(cadena) para mostrar el primer caracter de la cadena.
  • Use la instrucción F(cadena) para mostrar el caracter final de la cadena.
  • Use la operación cadena*2 para obtener el doble del total de caracteres dentro de la cadena.
  • Use la operación cadena/2 para obtener la mitad del total de caracteres dentro de la cadena. El resultado es redondeado.
